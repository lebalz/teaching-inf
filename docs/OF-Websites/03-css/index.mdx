# CSS

CSS ist eine Abkürzung für __Cascading Style Sheets__. Es handelt sich dabei um eine spezielle Sprache um die Darstellung von HTML-Elementen steuern zu können. Die Syntax gibt vor, wie die Elemente auf einer Webseite ausgewählt werden können. So kann das Aussehen ausgewählter Elemente verändert werden.

CSS ist modular aufgebaut. Die Struktur besteht aus einem oder mehreren Regeln, die aus __Selektoren__ und __Deklarationsblöcken__ bestehen.

## Element-Selektor

Ein Element-Selektor wählt alle Elemente mit demselben Tag-Namen aus und wendet die Style-Regeln auf diese an.

```css
p {
    color: red;
    background: blue;
}
```
Im obigen Beispiel werden alle Paragraphen `<p>` auf der Webseite ausgewählt und die Schriftfarbe auf rot und der Hintergrund auf blau gesetzt.
:::warning[Semikolon `;`]
Jede Deklaration endet mit einem Semikolon `;`. Also fast immer muss am Ende einer Zeile ein `;` stehen.
:::
