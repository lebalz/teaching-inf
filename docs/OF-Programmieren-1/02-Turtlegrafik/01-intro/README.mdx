Das erste Python-Programm zeichnet ein Quadrat. Damit die Befehle zum zeichnen verwendet werden k√∂nnen, muss zuerst eine Datei mit den entsprechenden Befehlen (ein sogenanntes Modul) `turtle` geladen werden.

:::note[`from turtle import *`]

Damit Python die Befehle zum zeichnen verwenden kann, m√ºssen diese zuerst aus einer Bibliothek **importiert** werden. Damit beim Importieren nicht jeder einzelne Befehl angegeben werden muss, wird das Zeichen `*` verwendet welches alle verf√ºgbaren Befehle importiert.
:::

:::note[Befehle: `forward`, `left`]

Alle Befehle in Python sind nach demselben Schema aufgebaut:

- einen Befehlsnamen
- runde Klammern
- _optional_ in den runden Klammern eine Auflistung\* von Parametern, die auch leer sein darf.

```

       \_____/  \
         /    Parameter (Bedeutung hier: 90 Schritte vorw√§rts)

       penup()
       \___/ \
         /   kein Parameter

```

\* werden mehrere Parameter √ºbergeben, dann werden diese mit Kommas abgetrennt, z.B. `goto(20, 30)`

Wird ein Befehl im Programm ausgef√ºhrt, spricht man von einem **Befehlsaufruf** - nebst dem Befehlsnamen geh√∂ren auch die runden Klammern dazu!
:::
