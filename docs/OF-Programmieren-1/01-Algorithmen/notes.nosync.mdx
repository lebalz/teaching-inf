# Notizen

## Didaktische Ideen:

- Determiniertheit: Verschiedenfarbgige Legosteine mitbringen, und alle SuS eine Figur bauen lassen. Dann die Anweisung geben, die Figur mit den Steinen in einer bestimmten Reihenfolge zu bauen (z.B. erst alle roten Steine, dann alle blauen, etc.). Dadurch wird das Konzept der Determiniertheit verdeutlicht.
