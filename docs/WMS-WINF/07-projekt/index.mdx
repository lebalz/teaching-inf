# Projekt

In diesem Semester werden wir ein Projekt durchführen, um die im Kurs behandelten Konzepte praktisch anzuwenden. Das Projekt wird in Zweierteams durchgeführt und resultiert jeweils in einer online verfügbaren Wordpress Seite. 

:::info[Ziel]
- Content Management System (CMS) verstehen und anwenden
- Wordpress als CMS nutzen und anpassen
- Eigenständiges Arbeiten und Problemlösungskompetenz fördern
:::

## Projektbeschreibung

Sie erstellen eine Marketing-Seite für ein neues Produkt mit Hilfe von Wordpress. Das Produkt kann bereits existierend sein oder von Ihnen konzipiert werden. Jegliche verwendeten Inhalte müssen aber von Ihnen erstellt werden und dürfen keine urheberrechtlich geschützten Inhalte enthalten. Die Seite soll alle relevanten Informationen über das Produkt enthalten und in einer ansprechenden Form präsentieren. Die Seite soll

- eine klare Zielgruppe ansprechen
- ein konsistentes Design aufweisen
- die Vorteile und Funktionen des Produkts hervorheben
- mehrere Varianten des Produkts präsentieren (auf mehreren Seiten verteilt)
- eine (fiktive) Kontaktmöglichkeit für Interessenten bieten
- weitere Anforderungen werden später eingeführt und bekannt gegeben
- ein Blog mit mindestens 3 Beiträgen enthalten, in dem über die Entwicklung des Produkts berichtet wird
