# HTML Links

HTML-Dokumente können auf andere HTML-Dokumente verweisen, zurückverweisen und weiterverweisen. Das Surfbare Internet wie wir es heute kennen ist im Endeffekt also eine Sammlung verlinkter HTML-Dokumente.


:::def[`<a href="pfad/zu/html">Link-Text</a>`]
Ein Link wird mit einem `<a>`-Tag erzeugt
- das Attribut `href` das Link-Ziel angibt
- der Inhalt des Tags als Link-Text angezeigt wird
:::

<HtmlIDE>
```html path=/index.html hideWarning
<h1>Startseite</h1>
Willkommen auf meiner Webseite!
<a href="./pages/first.html">Hier klicken</a> um zur nächsten Seite zu kommen.
```
```html path=/pages/first.html hideWarning
<h1>Seite 1</h1>
<ul>
    <li>
        <a href="./second.html">Zur nächsten Seite</a>
    </li>
    <li>
        <a href="../index.html">Zurück zur Startseite</a>
    </li>
</ul>
```
```html path=/pages/second.html hideWarning
<h1>Seite 2</h1>
<ul>
    <li>
        <a href="./first.html">Zur zu Seite 1</a>
    </li>
    <li>
        <a href="../index.html">Zurück zur Startseite</a>
    </li>
</ul>
```
</HtmlIDE>

<ProgressState id="6adae2c9-96b8-426a-b173-7499001765d0">
1. Öffnen Sie alle HTML-Seiten und flicken Sie allfällige Fehler bei den Bild-Links.
2. Fügen Sie eine Navigation zwischen den Seiten hinzu. Jede Seite soll **3 Links** haben:
    - "Zurück zur Startseite" um auf die Startseite kommen
    - "Weiter" um zur nächsten Seite zu kommen
    - "Vorherige" um zur vorherigen Seite zu kommen.

    Klickt man immer auf "Weiter", soll man zum Schluss wieder auf der Startseite landen, wobei die Reihenfolge Ihnen überlassen ist.
3. Fügen Sie die Navigations-Links in ein `<nav>` Tag ein.
4. (Freiwillig) Fügen Sie weitere passende Details, Bilder, Videos oder Sound zu den einzelnen Seiten hinzu.
