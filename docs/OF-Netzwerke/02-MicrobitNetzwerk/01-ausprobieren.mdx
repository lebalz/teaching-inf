## Ein erstes Netzwerk

Grösse
: 4-5 Micro\:Bits + Laptops
Ziel
: 4 __Client__-Geräte, eine "Verteilzentrale" (__Switch__) die miteinander kommunizieren können
Anforderungen
: Alle Geräte können miteinander kommunizieren
Geräte
: [Client](/network/client-L2/?power=1&group=10&address=1969383100)
: [Switch](/network/switch/?power=1&group=10&address=1969383100)
