# 3. Zeichencodierung

Datenpakete bestehen aus verschiedenen Zahlen und Zeichen. Wie lassen sich diese in Einsen und Nullen Гјbersetzen? Auf dem Computer ist jedem Zeichen eine eindeutige Zahl zugeordnet - eine sog. **Zeichencodierung**. Eine gГӨngige Zeichencodierung ist etwa [ASCII, 128 Zeichen](https://de.wikipedia.org/wiki/American_Standard_Code_for_Information_Interchange), wobei fГјr ein Zeichen genau `7 bit`, also 7 Stellen mit `0` oder `1`, notwendig sind. In diesen 128 mГ¶glichen Zeichen sind `33` Steuerzeichen und `95` normale Zeichen enthalten.

Steuerzeichen
: spezielle Zeichen wie bspw. das Zeichen `\n` fГјr einen Zeilenumbruch oder `\t` fГјr einen Tabulator.

Die ASCII-Codierung enthГӨlt noch keine Umlaute, weshalb mit [Latin 1, 256 Zeichen](https://de.wikipedia.org/wiki/ISO_8859-1) der Informationsgehalt um ein Bit erhГ¶ht wurde, so dass insgesamt `256` Zeichen codiert werden kГ¶nnen - die ersten `128` Zeichen sind dabei identisch mit ASCII.

::::def[`bin(<Zahl>)`]

## BinГӨrer Codierer

:::aufgabe

Schreiben Sie einen binГӨren Codierer, welcher einen Text ins binГӨre Гјbersetzt. Im vorgegebenen Code sehen Sie, wie Гјber jeden Buchstaben eines Wortes iteriert werden kann.


AufhГјbschen:
1. Das `0b` kann weggeschnitten werden, indem nur die Buchstaben *nach* dem zweiten Buchstaben verwendet werden:

    ```py live_py slim

    binaer = binaer[2:] # alles (:) nach dem zweiten Buchstaben wird verwendet
    print(binaer)
    ```
2. Jeder Buchstabe soll genau 1 Byte, also 8 Zeichen, umfassen. Dies erreicht man, wenn man mit Nullen (engl. *Zeros*) auffГјllt:
   
    ```py live_py slim
    binaer = '1011'
    binaer = binaer.zfill(8)
    print(binaer)
    ```
3. Statt immer eine neue Zeile zu verwenden, kann der `print`-Befehl auch angewiesen werden, am Ende ein Leerschlag einzufГјgen:

    ```py live_py slim
    for buchstabe in 'Hallo':
        print(buchstabe, end=' ')
    ```

<Solution id="646dc2f6-0af4-4bfc-bdb7-7e379dc4f08a">

```py live_py slim
for buchstabe in 'Hallo':
    zahl = ord(buchstabe)
    binaer = bin(zahl)
    # 0b wegschneiden
    binaer = binaer[2:]

    # auf 8 Stellen auffГјllen: Zero Fill
    binaer = binaer.zfill(8)
    print(binaer, end=' ') # Ausgabe: am Ende ein Leerschlag statt eine neuen Zeile einfГјgen

```
</Solution>
