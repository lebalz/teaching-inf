## Fibonacci
<Pyodide id="d5d60431-5589-4d38-9c8c-9b563194f5cc" title="Fibonacci ohne Rekursion">
```py
def fibonacci(n):
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    elif n == 2:
        return [0, 1]
    else:
        seq = [0, 1]
        for i in range(2, n):
            seq.append(seq[-1] + seq[-2])
        return seq
n = int(input("fib von? "))
print(fibonacci(n))
```
</Pyodide>

### Mit Memoization

<Pyodide id="22b0f649-9b77-4e97-9ccb-cfeb159bbde9" title="Fibonacci mit Memoization">
```py
from functools import lru_cache

@lru_cache
def fibonacci(n):
    if n in {0, 1}:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

for n in range(1, 200):
    print(f"Fibonacci({n}) = {fibonacci(n)}")
```
</Pyodide>

## Minimierung mit SciPy

<Pyodide id="8e484282-faac-4955-b0a2-b0a2e270f765" title="Minimierung mit SciPy">
```py
import numpy as np
from scipy import optimize
# Function to minimize: f(x) = (x - 3)^2
def func(x):
    return (x - 3) ** 2

# Minimize using scipy.optimize
result = optimize.minimize(func, x0=0)
print("Minimum at x =", result.x[0])
print("Function value at minimum =", result.fun)
```
</Pyodide>
